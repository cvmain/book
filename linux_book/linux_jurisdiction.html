# 1.3 更改文件或目录权限

linux中,每个文件和目录都有访问权限,用它来确定谁可以通过何种方式对文件进行访问和操作

有三种不同的用户可对文件或目录进行访问,分别是所有者,同组用户,其他用户;

并分别具有读、写、执行权限

# 1.3.1 更改权限的两种方法

  1. 文字设定法
chmod [who] [+|-|=] [mode] 文件名
1

  1. who可以是下面任意一个

    • "u"表示用户

    • "g"表示同组用户

    • "o"表示其他用户

    • "a"表示所有用户

  2. +|-|= 表示

    • +表示添加权限

    • -表示取消权限

    • =表示赋予权限

  3. mode是

    • r表示可读 对应4

    • w表示可写 对应2

    • x表示可执行 对应1

  4. 数字设定法

chmod [-R] 777 文件名
1
